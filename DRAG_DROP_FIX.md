# 拖放功能已修復！ 🎉

## 問題原因
拖放功能無法運作是因為 `tkinterdnd2` 套件雖然已安裝，但安裝到了不同的 Python 環境中。應用程式執行時找不到該套件。

## 解決方案
已建立專用的虛擬環境並正確安裝了所需套件：

✅ **已完成的修復：**
1. 建立了 Python 虛擬環境 (`.venv/`)
2. 在虛擬環境中安裝 `tkinterdnd2`
3. 更新了啟動腳本以使用虛擬環境
4. 改進了拖放事件處理程式（支援 macOS 格式）

## 現在如何啟動應用程式

### 方法 1: 使用啟動腳本（最簡單）
```bash
./run.sh
```

### 方法 2: 直接執行
```bash
.venv/bin/python batch_zip_gui.py
```

## 測試拖放功能

應用程式現在已經重新啟動並支援拖放！

**測試步驟：**
1. 在 Finder 中打開一個資料夾
2. 選擇一個或多個子資料夾
3. 將它們拖到 Batch ZIP 視窗中的列表區域
4. 放開滑鼠
5. 您應該會看到資料夾被加入，並在底部顯示成功訊息

## 拖放功能特色

✨ **支援多個項目**
- 一次可以拖放多個資料夾
- 自動過濾非資料夾項目

🛡️ **智慧處理**
- 自動偵測並略過重複的資料夾
- 顯示即時反饋訊息
- 支援 macOS 和 Windows 的拖放格式

## 如果您想在其他機器上使用

在新機器上執行：
```bash
./install_dependencies.sh
```

這會自動：
- 建立虛擬環境
- 安裝 tkinterdnd2
- 檢查並可選安裝 7-Zip
